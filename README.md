# 業務週報メーカー

毎日の記録テキストから「業務週報(人事部用)」を作成し、A4縦1枚のPDFに出力するアプリです。
サーバー不要・単一HTMLファイルで、`index.html` をブラウザ(Chrome推奨)で開くだけで使えます。

## 入手方法

1. このページの緑色の「**Code**」ボタン →「**Download ZIP**」でダウンロード
2. ZIPファイルを右クリック →「すべて展開」
3. 展開したフォルダの中の **index.html** をダブルクリック(Chromeで開きます)

## 使い方(3ステップ)

1. 毎日の記録を貼り付けて **「✨ AIで週報を自動作成」** を押す
   - AIがメモを**提出レベルの文章に清書**します: 職務内容の要約、成果の件数集計(例: 木52件+金10件→計62件)、観察の文章化、分析(→原因・打ち手)の作成まで
   - 「月」「火」…と曜日だけの行で日を区切った形式(`7/7(月)` の形式もOK)
   - 今週行動予定の平日には勤務時間・場所(9-18・資さん本社)が自動で入ります
2. シート上のセルを直接クリックして、気になるところだけ手直しする
3. **「🖨 PDFに保存(印刷)」** → 印刷ダイアログで「PDFに保存」を選択 → A4縦1枚のPDFが完成

氏名・対象週・入力内容・APIキーは自動保存されるので、毎週開いて貼り付けるだけです。

### AI清書の準備(初回のみ・1分)

1. https://aistudio.google.com/apikey を開き、Googleアカウントでログイン
2. 「APIキーを作成」を押してキーをコピー(無料枠で十分使えます)
3. アプリの「Gemini APIキー」欄に貼り付け(保存されるので次回から不要)

### APIキーを使わない方法

「📋 AI用の指示文をコピー」→ 普段お使いのGeminiやChatGPTの画面に貼って送信 →
返ってきたJSONをアプリに貼り付けて「取り込み」。仕上がりは同じです。

### 簡易作成(AIなし)

AIを使わずキーワード抽出だけで下書きする「簡易作成」ボタンもあります(精度は下がります)。
- **成果**: 「完了」「済み」「実施」「○件」などを含む文 / **観察**: 「〜しかない」「残念」「時間がとられる」など
- **その他**: 「依頼」「待ち」「継続して確認」など。5枠を超えた分は最後の枠にまとめます

## 機能

- **自動保存**: 入力内容はブラウザ(localStorage)に自動保存。閉じても消えません
- **書き出し/読み込み**: JSONファイルとして保存・復元(週ごとのアーカイブに)
- **1枚判定**: 内容が多くA4 1枚に収まらない恐れがある場合は警告を表示
- **合計時間**: 「時間」列の合計を自動計算

## 記録テキストの書き方(例)

```
月
新入社員　入社式　予行演習実施のあと本番　滞りなく
書類記入　オリエンテーション
火
引継ぎ詳細
クルー採用の打ち手まとめ
```

「サンプル」ボタンで記入例を確認できます。
